#About

The Neuralogix Company Website serves as the official online presence for Neuralogix, showcasing our services and company information. The website is built with Tailwind CSS to ensure a fully responsive and modern design, optimized for performance and accessibility.

#Features

Fully responsive design for desktop, tablet, and mobile

Tailwind CSS utility-first styling

Clean and modern UI components (hero section, services,  footer)

Easy customization and scalability
